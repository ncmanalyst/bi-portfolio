# Business Intelligence Portfolio

This repository contains BI dashboards designed to transform data into
clear, actionable insights for decision-making.

---

## Project 1: Financial Operations Dashboard

### Overview
A financial operations dashboard built to monitor cash flow, liquidity,
expenses, and burn rate in a startup or small business environment.

### Business Questions
- What is the current cash position?
- Is the burn rate sustainable?
- How is liquidity distributed across accounts?
- Where are the main operational expenses?

### Key Metrics
- Net Cash Flow
- Cash Position
- Burn Rate
- Liquidity by Account
- Recent Transactions
- Expense Distribution

### Insights
- Identifies liquidity concentration risk
- Highlights spending structure and cost allocation
- Supports short-term cash planning

### Skills Demonstrated
- Financial analytics
- KPI design
- Dashboard storytelling
- Business-focused data visualization

---

## Project 2: Personal Health & Activity Analytics Dashboard

### Overview
A behavioral analytics dashboard focused on personal activity, recovery,
and habit optimization through data-driven insights.

### Analytical Questions
- How does activity level affect sleep quality?
- Is sedentary time impacting recovery?
- Are daily habits aligned with healthy benchmarks?

### Key Metrics
- Daily Steps
- Calories Burned
- Sleep Quality Score
- Activity Distribution

### Insights
- High sedentary time correlates with lower sleep quality
- Moderate activity dominates daily movement
- Activity levels exceed minimum health recommendations

### Skills Demonstrated
- Behavioral data analysis
- Pattern recognition
- Insight generation
- KPI interpretation
